# E-Commerce Sales Analysis Dashboard (Power BI)

This project presents an interactive Power BI dashboard analyzing e-commerce sales data.

## Key Metrics
- Total Sales
- Total Profit
- Total Products Sold

## Visualizations
- Top Products by Sales
- Sales by Category
- Monthly Sales Trend
- Regional Sales Distribution

## Filters
- Region Filter
- Category Filter

## Tools Used
- Power BI Desktop
- Data Visualization
- Business Analytics

## Dataset
E-commerce sales dataset used for analysis.
